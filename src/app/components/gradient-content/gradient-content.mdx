| Attribute      | Description                                             | Type             | Accepted values          | Default                                             |
| -------------- | ------------------------------------------------------- | ---------------- | ------------------------ | --------------------------------------------------- |
| gradient       | Gradient in css format for background style             | `string`         | -                        | `linear-gradient(to bottom, accents_1, background)` |
| img            | path to a image which will be lazy loaded as background | `string`         | -                        | -                                                   |
| gradientheight | the height of the gradient to the relative element      | `string`         | -                        | `50vh`                                              |
| ...            | native props                                            | `HTMLAttributes` | `'id', 'className', ...` | -                                                   |

</Attributes>
