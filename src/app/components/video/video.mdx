# Video

The `Video` UI component provides a versatile way to display video content on your website or application.

<Playground
  scope={{ Video }}
  title={"Example of a embed video"}
  desc="You can use the `Video` component to embed video content in your web pages. It supports various video formats and offers options for customization."
  code={`
<Video
  src="https://media.w3.org/2010/05/sintel/trailer_hd.mp4"
  controls={true}
  autoplay={true}
  muted={true}
/>
`}
/>
<Attributes edit="/app/components/video.mdx">
  <Attributes.Title>Video.Props</Attributes.Title>

| Attribute    | Description                              | Type      | Accepted Values            | Default |
| ------------ | ---------------------------------------- | --------- | -------------------------- | ------- |
| **src**      | The source URL of the video.             | `string`  | -                          | -       |
| **controls** | Determines if video controls are shown.  | `boolean` | `false` (to hide controls) | `true`  |
| **poster**   | Poster image for video.                  | `string`  | -                          | -       |
| **loop**     | Play video in loop.                      | `boolean` | `true` to start loop       | `false` |
| **autoplay** | Determines if the video should autoplay. | `boolean` | `true` (to autoplay)       | `false` |
| **muted**    | Determines if the video should be muted. | `boolean` | `true` (to mute)           | `false` |

</Attributes>
